# Gene Variant Search

## Development

- Download `git clone https://github.com/calvinyhu/gvs.git`.
- Navigate to `/gvs`.
- Install by running `npm run installAll`.
- Start the app by running `npm run dev`.

## Testing

- In the root folder run `npm test`.
- In `/client` run `npm test`.
